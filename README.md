  - [x] Share on social medias (Twitter, WhatsApp...)
- [x] Random Color
  - [x] Keyboard shorcuts
- [x] Color picker
- [x] Password generator
- [ ] Translate
- [ ] Barcode Reader
- [ ] File converter (images, videos, audios, documents)
- [ ] Video downloader (at least YouTube)
